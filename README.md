# Claude Skills

Reference files for Claude.ai and Claude Code skills.

## Structure
Each folder = one skill with SKILL.md and optional references/

## Usage
- Claude.ai: install SKILL.md via Claude settings, fetch references from raw GitHub URLs
- Claude Code: reads locally from ~/workspace/claude-skills/

## Update
cd ~/workspace/claude-skills && git pull

- stop-slop/ - Remove AI writing patterns from prose

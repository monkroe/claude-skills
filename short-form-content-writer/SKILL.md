# Short-Form Content Writer

The final step: the words that ship the video AND the structure that makes it
spread. A flawless AI clip still flops with a weak hook, no retention design,
or a vague caption. This skill writes the hook, the on-screen text, the script,
the description, and the CTA — all built around how short-form algorithms
actually distribute content. Output is clean, copy-paste English, mobile-native.

Ask what the user needs if unclear. Default for a finished clip: hook options +
on-screen text + script + TikTok description + Instagram caption + CTA.

## How short-form virality actually works (read this first)

Every piece of copy below serves one of the metrics the algorithm rewards.
Knowing the mechanic tells you what to write:

- **Watch time / completion rate is king.** The platform pushes videos people
  finish (or re-watch). Everything in the script exists to keep the viewer to
  the last frame. A 15-20s video that gets watched fully beats a 60s video
  people drop at 10s.
- **Re-watches multiply reach.** A clip that loops seamlessly or rewards a
  second viewing ("wait, what?") racks up plays-per-viewer. Design the last
  line to loop back into the first.
- **Saves and shares are the strongest reach levers.** A comment is good; a
  save ("I'll need this later") and a share ("you have to see this") tell the
  algorithm the content has lasting value. AI tutorials and "how did they do
  that" clips are inherently saveable — lean into it.
- **Comments compound early.** Replies and comment threads in the first hour
  signal a conversation. Engineer one easy, irresistible thing to comment.
- **The first 1-2 seconds decide everything.** Most viewers leave in the first
  3 seconds. The hook (spoken + on-screen text + the opening frame) has to land
  before the brain decides to scroll.

The realism angle is a built-in virality engine: a believable face that turns
out to be AI creates the "is this real?" snag that drives both comments
(guesses) and shares (disbelief). Name the realism, then break it.

## Retention mechanics to build into every script

These are the levers that hold watch time. Apply them, don't just list them:

- **Open loop in the first line.** Promise something that only resolves at the
  end: "The last clip is the one nobody believes." The unanswered question
  holds the viewer.
- **Show the payoff frame first, then earn it.** Cold-open on the most
  jaw-dropping moment for half a second, then cut to "here's how." The viewer
  stays to reach what they already glimpsed.
- **No dead air, no warm-up.** Cut the intro. Every line earns the next. If a
  line can be removed and the clip still makes sense, remove it.
- **Pattern interrupts every few seconds.** Change the shot, the angle, the
  text, or the pace often enough that the eye never settles into "I've seen
  enough."
- **Stack value, build to the best.** Order the steps/reveals so the most
  impressive is last — the reason to stay.
- **Seamless loop close.** Write the final line so it flows back into the hook,
  turning one view into two or three.

## 1. Hooks

The hook is the first 1-2 seconds (spoken AND on-screen text) plus the
caption's first line. Always write **6-8 options** so the user can pick. Keep
each under ~10 words — punchy beats complete.

Realism hook banks (adapt the specifics):

- **Disbelief**: "This person doesn't exist." / "She's not real. Look closer."
- **Provocation / hot take**: "Photography is over." / "You can't trust a photo
  anymore." Mild controversy invites comments.
- **Tool vs skill**: "The tool isn't the secret. This is." / "Everyone has the
  same AI. Not the same results."
- **Before/after**: "One photo. Then I made her talk." / "From a still image to
  a real person."
- **Numbered promise**: "3 steps to a face that never changes." / "Same person,
  5 different videos."
- **Open-loop tease**: "Wait for the last clip." / "The third one fooled my
  own mom."
- **Callout / question**: "Can you tell this is AI?" / "Real or generated?
  Comment your guess."

Match the hook to the clip's single strongest moment. If the wow is consistency
across shots, lead with "same person"; if it's one hyperreal face, lead with
disbelief; if there's a build, use an open loop and pay it off at the end.

## 2. On-screen text (title overlay)

Short-form is watched on mute as often as not, so the opening on-screen text is
a second hook that must carry the video alone. Write a 3-6 word overlay for the
first frame, plus optional step labels ("Step 1: the face") that double as
retention anchors. Keep it big, short, and high-contrast in intent. The
on-screen hook can differ from the spoken hook to cover both the muted and
sound-on viewer.

## 3. Reel / TikTok script (15-60s)

Structure as beats, not paragraphs. Map each spoken line to what's on screen
(screen cues in brackets) and apply the retention mechanics above.

1. **Cold open / hook (0-2s)** — the jaw-dropping frame + spoken hook + on-
   screen text, ideally with an open loop. `[show the most impressive frame
   first]`
2. **Setup (2-5s)** — the tension/promise in one line: what they'll see and why
   it matters.
3. **Payoff / steps (5-40s)** — the build, one line per beat, each mapped to a
   visual, most impressive last. For realism content: anchor face → angles →
   video → the "same person" reveal.
4. **Climax (last 3-5s)** — deliver the open-loop promise; restate the result
   with the contrast: "From a white-background photo to a talking person. Same
   face the whole time."
5. **Loop + CTA (final line)** — one ask, phrased so it also loops back to the
   hook.

Keep spoken lines short enough to actually say in time — a 20s Reel is ~6-9
short lines. Faster is usually better; trim ruthlessly.

## 4. Descriptions / captions

Platform-specific — not interchangeable.

**TikTok** — 1-2 short lines, casual, front-load a searchable keyword (TikTok
is a search engine now), 3-5 hashtags mixing one broad + a few niche. The first
caption line is a second hook; make it pull. A subtle question in the caption
invites a comment.

**Instagram** — fuller: a hook line, 1-2 lines of value/context that make it
worth saving, the CTA, then a separate hashtag block (5-15, broad + niche) with
line breaks between blocks. Captions that teach or promise a resource earn
saves, which Instagram rewards heavily.

Keep both fluff-free. Every line pushes toward watching, saving, sharing, or
commenting.

## 5. CTAs (matched to the reach lever you want)

One ask per post — splitting attention lowers all of them. Pick to the goal:

- **Saves (strong reach)** → "Save this before you forget the steps." /
  "Save it — you'll want this workflow later."
- **Shares (strongest reach)** → "Send this to someone who swears they can spot
  AI." / "Tag the friend who won't believe this is fake."
- **Comments (early signal)** → "Comment [KEYWORD] and I'll send the steps." /
  "Real or AI? Comment 1 or 2." Binary, low-effort, or keyword bait converts
  best. A keyword CTA also feeds DM automation.
- **Follows (audience growth)** → "Follow — this is part 1 of the build." Tie
  the follow to a concrete next payoff (a series).

Make it specific and low-effort: "Comment YES" beats "let me know your
thoughts."

## 6. Growth tactics beyond a single post

When the user wants more than one clip, suggest these:

- **Series / open loops across posts.** "Part 1 of building an AI influencer"
  trains the audience to follow for part 2. Cliffhanger the end.
- **Batch one win into many angles.** One good character → a face reveal, a
  "guess if it's real", a step-by-step, a fails/bloopers cut. Same asset,
  multiple hooks.
- **Ride formats and sounds, not just topics.** Plug the realism reveal into a
  trending format or audio so the algorithm's existing momentum carries it.
  (The user supplies the current trend; the skill adapts the copy to it.)
- **Post consistently.** Volume + consistency beats one perfect post; the
  algorithm rewards accounts that show up. Encourage a repeatable cadence.
- **First-hour engagement.** Reply to every early comment fast — it stacks the
  comment signal when it matters most.

## Common mistakes to avoid

- Burying the wow-moment instead of cold-opening on it.
- A slow intro / warm-up — kills the 3-second window.
- Two CTAs in one post — pick one.
- Hashtag soup of only broad tags — mix in niche.
- Explaining instead of revealing — let the clip prove it; copy points at it.
- No loop and no open loop — the two cheapest ways to multiply watch time.
- Sounding like an ad — stay first-person and casual.

## Delivery

Deliver requested pieces clearly labeled. Hooks as a numbered list. On-screen
text as short labeled overlays. Script as labeled beats with screen cues in
brackets. Each caption in its own code block so it's copyable. English,
mobile-native, no filler.

## Worked mini-example

Clip: a consistent AI character going from a still photo to a talking video.

Hooks:
1. This person doesn't exist.
2. One photo. Then she started talking.
3. Same face. Five different videos.
4. Wait for the last clip.
5. Real or AI? Look closer.

On-screen text (frame 1): `SHE ISN'T REAL`  · step labels: `the face` →
`every angle` → `now she talks`

Script (~20s):
- `[cold open on the talking close-up]` "This face has never existed." (open
  loop: how?)
- `[white-background anchor]` "It started as one photo."
- `[angle sheet]` "Then I generated her from every angle."
- `[video clip]` "Then I made her move and talk."
- `[two clips side by side]` "Same person in every shot. That's the hard part."
- "Comment FACE and I'll send the steps." `[last frame matches the opening —
  loops]`

TikTok caption:
```
Made this face from one photo, then animated it. Same person every time 👀
real or ai? #aivideo #aicharacter #aiart
```

Instagram caption:
```
This person doesn't exist.

One photo for the face. One reference sheet for the angles. Then image-to-video
to make her move — same identity in every clip.

Save this so you don't lose the workflow. Comment FACE and I'll walk you
through it.

#aivideo #aicharacter #aiinfluencer #aiart #consistentcharacter
```

## When NOT to use this skill

- Making the character, images, or video → the image and video skills.
- Long-form scripts (full YouTube tutorials) — this skill is built for short-
  form; adapt the beat structure but keep it tight.
